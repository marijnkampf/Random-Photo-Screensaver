    Image rotation

    Use following keys to rotate images(s)
    [ rotate image 270° CW
    ] rotate image 90° CW
    + rotate image 180°
    Please note: if filename.bak exists it isn't overwritten.
    For multi monitors: select monitors with 1 ... 9 or 0 for all.
