    Filter content

    The following dimensions can be used in filters:
    • [database column] [description]
    • paths full path (folder and filename)
    • parent paths folder path only
    • filenames filename (including extension)
    • created created date
    • modified modified date
    • metadata all metadata of media; each line contains metadata name tab (\t) metadata value
    • size file size
    • width width of the media in pixels
    • height height of the media in pixels
    • area pixel width x height of the media

  • Metadata

    Quick metadata

    See ExifTool TagNames for EXIF/IPTC and other meta tag names that may be used.
    Press Ctrl + C when screensaver is running to copy available meta tag names and metadata to clipboard (close configuration screen first).
    metadata iptc xmp exif

    ExifTool

    Image rotation

    Please note: if filename.bak exists it isn't overwritten.
    Use following keys to rotate images(s)
    [ rotate image 270° CW
    ] rotate image 90° CW
    + rotate image 180°

    Select monitors with 1 ... 9 or 0 for all.
    turning rotate rotating

    Image rating

    Please note: if filename.bak exists it isn't overwritten.
    Use Control + 0 ... 5 keys to rate images(s)

    Select monitors with 1 ... 9 or 0 for all.
    add starring make favourite rate
